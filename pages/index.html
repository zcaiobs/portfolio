The Stack

Hi, I'm Caio and I'll talk a little bit about my stack, HTML5, CSS3, Javascript, NodeJS and MongoDB. The basis for creating multiple applications is present in these technologies. Perfect for creating any web solution. This stack allows us to create applications and prototypes in a short time and at a low cost. What made me choose these technologies was the large community, because I can easily find all kinds of documentation, books, tutorials and lectures about it.

Knowing how to consume an api is essential today, as many applications are built on the standard Rest (Representational State Transfer), is a style of software architecture that defines a set of restrictions to be used when creating web services. Api rest is a way to ensure that the web services will have scalability and availability.
